Installation
------------

Install universal wheel

.. code:: bash

    $ pip install CRC-ITU


Install with cython extension

.. code:: bash

    $ pip install cython
    $ pip install --no-binary :all: CRC-ITU


Usage
-----

.. code:: python

    from crc_itu import crc16
    print(hex(crc16(b'your bytes')))
